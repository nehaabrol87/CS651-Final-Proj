    About Us

  • Website This website gives you following 5 options
    Calculate Daily meal plan for family from all food groups
    Plan your diet for next day and track progress
    Calculate Daily physical activity requirement for all age groups
    Plan your activity for next day and track progress
    Calculate BMI
  • Pages/routes
    Home
    User needs to fill the sign Up form to create Profile.Email would be sent to activate user Profile
    User logs in after verifying email account and is redirected to profile page
    Profile
    User needs to fill mandatory information before he can access his above 5 options
    User can click on any of the 5 options mentioned above
    User can track progress of his past diets/activity
    User can email himself plan for tomorrow
  • Misc This website provides following extra functionality
    User can expand drawer on profile/homepage to read useful health tips
    Web Application service sends the user an emal each morning at 9 am to remind him to log plan for tomorrow

    Technical

  • Client Side
    • Angular 1.4.0-rc.1
    • TypeScript 1.5.0-alpha
    • Angular2 Router / UI-Router
    • Gulp
    • Jspm for package management
    • (Node modules are only used for gulp plugins)
    • Express.js - Used as a development server with proxy
  • Server Side
    • WCF service for server side code(Host on IIS) http://localhost/finalservice/Service.svc/
    • Global.asax file to allow cross domain requests(Node JS on 8080)
    • Web application service for sending emails daily at 9am(reminders) I was running into installation issues so was not able to test this
  • Database
    • HealthyHumans Database has following 3 tables
    • Users(User Profile)
    • Meals(Meal information for user)
    • Activity(Activity information for user)
